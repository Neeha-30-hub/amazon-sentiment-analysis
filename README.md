# Amazon Product Reviews Sentiment Analysis

## Project Overview
Sentiment Analysis on Amazon Product Reviews using NLP and Machine Learning.

## Dataset
- Source: Kaggle Amazon Reviews Dataset
- Size: 50,000 reviews
- Labels: Positive and Negative

## Steps
1. Data Loading and Preprocessing
2. Text Cleaning
3. TF-IDF Vectorization
4. Logistic Regression Model
5. Model Evaluation

## Results
- Accuracy: 88%
- Positive Reviews: 25,506
- Negative Reviews: 24,494

## Key Findings
- Model correctly identifies positive and negative reviews
- TF-IDF with Logistic Regression gives 88% accuracy

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- NLP (TF-IDF)
